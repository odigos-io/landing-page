We’re excited to announce the public beta release of the **new Odigos UI**, a transformative update that makes observability smoother, faster, and more intuitive than ever before.

To get started now, run the UI with the beta flag:

```bash
bash
Copy code
odigos ui  --beta
```

Want to learn more about what’s new? Read on!

### **What’s New in the Odigos Beta UI?**

This release has been in the works for a while, and we’re proud to share these highlights:

1. **Modern, User-Friendly Design:** Navigate effortlessly with a fresh, streamlined layout.
2. **Enhanced Observability Features**
   - **Instrumentation rules**: Control how telemetry is recorded from your application
   - **Refined Overview Page**: Get a bird’s-eye view of your entire system:
     • Real-time metrics and status indicators
     • Quick access to detailed logs and traces
   - **Actions**: Modify the OpenTelemetry data recorded by Odigos Sources, before it is exported to your Odigos Destinations.
3. **Built for the Future**

   This beta sets the foundation for even more powerful features coming soon!

### **A Work in Progress**

As this is a beta version, you might notice some features still under development or subject to change. Your feedback is essential to help us shape the final product!

Try it now and be part of the journey!
